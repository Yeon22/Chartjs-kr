    Interactions

    The hover configuration is passed into the options.hover namespace. The global hover configuration is at Chart.defaults.hover. To configure which events trigger chart interactions, see events.

    NameTypeDefaultDescription
    modestring'nearest'Sets which elements appear in the tooltip. See Interaction Modes for details.
    intersectbooleantrueif true, the hover mode only applies when the mouse position intersects an item on the chart.
    axisstring'x'Can be set to 'x', 'y', or 'xy' to define which directions are used in calculating distances. Defaults to 'x' for 'index' mode and 'xy' in dataset and 'nearest' modes.
