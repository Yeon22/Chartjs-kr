폰트

폰트

차트의 모든 폰트를 바꿀 수 있는 4개의 특별한 전역 설정이 있습니다. 이 옵션들은 Chart.defaults.global 안에 있습니다. 전역 폰트 설정은 오직 설정에 포함되어있지 않은 더 특정한 옵션들만 적용할 수 있습니다.

예를 들어, 이 차트에서 범례 안의 래이블들을 제외한 텍스트는 모두 빨간색이 될 것입니다. 

Chart.defaults.global.defaultFontColor = 'red';
let chart = new Chart(ctx, {
    type: 'line',
    data: data,
    options: {
        legend: {
            labels: {
                // 이 더 특정한 폰트 속성은 전역 속성을 덮어씁니다
                fontColor: 'black'
            }
        }
    }
});
이름 형식 기본 값 설명
defaultFontColor Color '#666' 모든 텍스트의 기본 색상을 지정합니다.
defaultFontFamily string "'Helvetica Neue', 'Helvetica', 'Arial', sans-serif" 모든 텍스트의 기본 폰트 패밀리를 지정합니다.
defaultFontSize number 12 텍스트의 기본 폰트 크기(픽셀 단위)를 지정합니다. 방사선형 스케일 포인트 래이블엔 적용되지 않습니다.
defaultFontStyle string 'normal' 기본 폰트 스타일을 지정합니다. 툴팁 제목이나 꼬리말에는 적용되지 않습니다. 차트 제목에는 적용되지 않습니다.

없는 폰트

만약 폰트가 차트에 명시되어있는데 시스템에 존재하지 않는다면, 브라우저는 준비될때까지 폰트를 적용하지 않을 것입니다. 만약 차트에 이상한 폰트가 나타난다면, 시스템에 존재하는 폰트를 적용했는지 확인해보세요. 이슈 3318에서 더 자세한 사항을 확인하세요.
