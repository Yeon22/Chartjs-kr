상호작용

호버 설정은 options.hover 네임스페이스의 일부입니다. 전역 호버 설정은 Chart.defaults.global.hover에서 가능합니다. 차트 상호작용 이벤트 트리거에 대한 설정들은 이벤트에서 보실 수 있습니다.

이름 유형 기본 값 설명
mode string 'nearest' 툴팁에서 보여질 엘리먼트를 설정합니다. 상호작용 모드 에서 자세한 사항을 볼 수 있습니다.
intersect boolean true 만약 true라면, 마우스 위치가 차트 아이템을 가로지를 때만 호버 모드가 적용됩니다.
axis string 'x' 거리 계산에 사용하기 위해 'x', 'y', 또는 'xy' 방향들을 정의하여 설정할 수 있습니다. 기본적으로 'index' 모드에선 'x' 이고 dataset 과 'nearest' 모드에선 'xy'입니다.
animationDuration number 400 호버 스타일 변경에 애니메이션을 적용하는 시간(밀리 초)입니다.
