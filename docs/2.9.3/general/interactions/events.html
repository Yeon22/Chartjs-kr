이벤트

이벤트

다음 속성들은 차트가 이벤트 상호작용을 하기위한 방법을 정의합니다.

이름 유형 기본 값 설명
events string[] ['mousemove', 'mouseout', 'click', 'touchstart', 'touchmove'] events 옵션 정의는 차트가 감지해야하는 툴팁과 호버링 브라우저 이벤트를 정의합니다. 더보기...
onHover function null 이벤트가 발생하면 호출됩니다. 차트의 컨텍스트에서 호출되고 이벤트와 활성화 요소들의 배열을 넘깁니다 (막대, 포인트 등).
onClick function null 이벤트가 'mouseup' 또는 'click' 유형 이벤트이면 호출됩니다. 차트의 컨텍스트에서 호출되며 이벤트와 활성화 요소들의 배열을 넘깁니다.

이벤트 옵션

예를 들어, 차트가 오직 클릭 이벤트에 대한 반응만 가진다면, 이렇게 할 수 있습니다: 

var chart = new Chart(ctx, {
    type: 'line',
    data: data,
    options: {
        // 이 차트는 마우스 오버 등에 대한 반응은 하지 않을 것입니다
        events: ['click']
    }
});
