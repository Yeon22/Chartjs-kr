시작하기

Chart.js를 시작해봅시다!

먼저, 페이지에 캔버스가 필요합니다.

<canvas id="myChart"></canvas>

현재 사용할 수 있는 캔버스가 있습니다. 페이지에 Chart.js를 받습니다. 

<script src="https://cdn.jsdelivr.net/npm/chart.js@2.8.0"></script>

이제 차트를 만들 수 있습니다. 페이지에 스크립트를 추가합니다:

var ctx = document.getElementById('myChart').getContext('2d');
var chart = new Chart(ctx, {
    // 만들기 원하는 차트의 유형
    type: 'line',

    // 데이터 집합을 위한 데이터
    data: {
        labels: ['January', 'February', 'March', 'April', 'May', 'June', 'July'],
        datasets: [{
            label: 'My First dataset',
            backgroundColor: 'rgb(255, 99, 132)',
            borderColor: 'rgb(255, 99, 132)',
            data: [0, 10, 5, 2, 20, 30, 45]
        }]
    },

    // 설정은 여기서 하세요
    options: {}
});

Chart.js를 쉽게 사용했습니다! 여기서부터 차트를 커스터마이징하는데 도움을 주는 스케일, 툴팁, 라벨, 색상, 커스텀 반응, 그리고 더 많은 옵션을 탐험할 수 있습니다.

모든 예시들은 온라인으로 제공하지만 모든 릴리즈에 첨부된 Chart.js.zip 보관소를 다운로드하여 /samples 폴더에서 로컬로 샘플을 경험할 수 있습니다.
